│   ├── shelves.jpg     # תמונה לתליית מדפים

## הוראות התקנה ב-GitHub Pages

### שלב 1: יצירת Repository
1. היכנס ל-GitHub ויצור repository חדש
2. תן לו שם כמו `handyman-website` או `tzvi-handyman`
3. סמן אותו כ-Public

### שלב 2: העלאת הקבצים
1. העלה את כל הקבצים ל-repository:
   - index.html
   - style.css
   - script.js
   - תיקיית images/ עם כל התמונות שלך

### שלב 3: הפעלת GitHub Pages
1. לך ל-Settings של ה-repository
2. גלול למטה ל-"Pages" בתפריט הצד
3. תחת "Source" בחר "Deploy from a branch"
4. בחר את ה-branch הראשי (main או master)
5. בחר "/" (root) כתיקיית המקור
6. לחץ Save

### שלב 4: גישה לאתר
האתר שלך יהיה זמין בכתובת:
```
https://<username>.github.io/<repository-name>/
```

לדוגמה: `https://tzvi.github.io/handyman-website/`

## עדכון מספר טלפון

כאשר תקבל מספר טלפון קבוע:
1. פתח את `index.html`
2. חפש את `tel:0500000000`
3. החלף ב-`tel:05XXXXXXXX` (המספר שלך ללא מקפים)
4. חפש את `050-000-0000`
5. החלף בפורמט הנוח לקריאה: `050-XXX-XXXX`

## התאמות אישיות

### שינוי צבעים
ב-`style.css` תוכל לשנות את הצבעים:
- שורה 24-25: צבעי הגרדיאנט בהדר
- שורה 54: צבע כפתור ה-CTA (ירוק כרגע)
- שורה 195: צבע כותרות השירותים (סגול כרגע)

### הוספת קרוסלת חבילות
כאשר תהיה מוכן להוסיף את הקרוסלה:
1. פתח את `index.html`
2. מצא את הקטע: `<section class="carousel-placeholder">`
3. החלף בקוד הקרוסלה שתרצה

## תחזוקה ועדכונים

- כדי לעדכן מחירים: ערוך את `index.html` בטבלאות המחירים
